# Minecraft-Utility-GrimDisablerMeteor

Addon o módulo orientado a **Meteor Client**, enfocado en desactivar o evadir ciertos chequeos de **Grim** usando una mecánica basada en trident spam.

## Qué contiene

- **build.gradle**: configuración de compilación.
- **gradle.properties** y `settings.gradle`: ajustes del proyecto.
- **gradlew** y `gradlew.bat`: wrappers de Gradle.
- **README.md**: documentación del repositorio.
- archivos de configuración como `.editorconfig` y `.gitignore`.

## Enfoque del proyecto

El repositorio está pensado como módulo específico para Meteor Client en una versión concreta de Minecraft, con foco en comportamiento técnico ligado a Grim.

## Cómo compilar

```bash
./gradlew build
```
